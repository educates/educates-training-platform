---
title: "Deprecated: Expose Dashboard"
---

The `dashboard:expose-dashboard` action is the original format for opening an
existing dashboard tab. It has been superseded by `dashboard:open-dashboard`
which is functionally identical. The `dashboard:expose-dashboard` format is still
supported for backward compatibility but should not be used in new workshops.

## Basic Expose Dashboard

The markdown for this action is:

~~~markdown
```dashboard:expose-dashboard
name: Example
```
~~~

Click the action below to test it:

```dashboard:expose-dashboard
name: Example
```
