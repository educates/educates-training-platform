---
title: Workshop Overview
---

This workshop tests the `files:` group of clickable actions provided by
Educates. These actions allow workshop users to download files from the
workshop session, copy file contents to the clipboard, and upload files to the
workshop session.

The following features are covered:

**File download (`files:download-file`):**

* Basic file download with `path` property
* Custom download name with `download` property
* File preview with `preview` property

**File copy (`files:copy-file`):**

* Copy file contents to clipboard with `path` property
* Copy with preview using `preview` property

**File upload (`files:upload-file` and `files:upload-files`):**

* Upload a single named file
* Upload multiple arbitrarily named files
