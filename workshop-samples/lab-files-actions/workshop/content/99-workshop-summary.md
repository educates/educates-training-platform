---
title: Workshop Summary
---

This workshop tested the following `files:` clickable action features:

**File download (`files:download-file`):**

* Basic file download using `path` property
* Custom download name using `download` property
* Data variables in download name
* File content preview using `preview` property
* Combined custom name and preview

**File copy (`files:copy-file`):**

* Copy file contents to clipboard using `path` property
* Copy with file content preview using `preview` property

**File upload (`files:upload-file`):**

* Upload a single file with a specified name using `path` property

**File upload (`files:upload-files`):**

* Upload multiple files with original names preserved
