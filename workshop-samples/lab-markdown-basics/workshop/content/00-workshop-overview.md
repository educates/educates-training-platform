title: Workshop Overview
---

This workshop demonstrates common markdown formatting as rendered by Educates.
It is intended as a visual test and reference for verifying that markdown
elements are styled and displayed correctly.

The following topics are covered across the workshop pages:

- **Headings** — Heading levels from `h1` through `h6`
- **Text Formatting** — Bold, italic, strikethrough, and inline code
- **Lists** — Unordered, ordered, nested, and task lists
- **Links and Images** — Inline links, reference links, and images
- **Code Blocks** — Fenced code blocks with syntax highlighting
- **Blockquotes and Horizontal Rules** — Nested blockquotes and dividers
- **Tables** — Tables with column alignment
- **Admonitions** — Note, warning, and danger shortcodes

Each page shows the markdown source used alongside the rendered result so that
workshop authors can use this as a reference.
