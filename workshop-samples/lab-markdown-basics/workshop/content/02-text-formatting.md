---
title: Text Formatting
---

Markdown provides several inline formatting options for styling text.

## Bold

The markdown for bold text is:

~~~markdown
This is **bold text** and this is also __bold text__.
~~~

This is **bold text** and this is also __bold text__.

## Italic

The markdown for italic text is:

~~~markdown
This is *italic text* and this is also _italic text_.
~~~

This is *italic text* and this is also _italic text_.

## Bold and Italic

The markdown for bold and italic text together is:

~~~markdown
This is ***bold and italic*** text.
~~~

This is ***bold and italic*** text.

## Strikethrough

The markdown for strikethrough text is:

~~~markdown
This is ~~strikethrough~~ text.
~~~

This is ~~strikethrough~~ text.

## Inline Code

The markdown for inline code is:

~~~markdown
Use the `echo` command to print text. Escape a backtick with `` `\` ``.
~~~

Use the `echo` command to print text. Escape a backtick with `` `\` ``.

## Combining Formats

The markdown for combined formatting is:

~~~markdown
You can **nest _italic_ inside bold** or use `code` alongside **bold text**.
~~~

You can **nest _italic_ inside bold** or use `code` alongside **bold text**.
