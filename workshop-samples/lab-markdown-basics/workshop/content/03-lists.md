---
title: Lists
---

Markdown supports unordered, ordered, and task lists. Lists can also be nested
to create sublists.

## Unordered Lists

The markdown for an unordered list is:

~~~markdown
- First item
- Second item
- Third item
~~~

- First item
- Second item
- Third item

## Nested Unordered Lists

The markdown for nested unordered lists is:

~~~markdown
- Fruit
  - Apple
  - Banana
- Vegetables
  - Carrot
  - Broccoli
~~~

- Fruit
  - Apple
  - Banana
- Vegetables
  - Carrot
  - Broccoli

## Ordered Lists

The markdown for an ordered list is:

~~~markdown
1. First step
2. Second step
3. Third step
~~~

1. First step
2. Second step
3. Third step

## Nested Ordered Lists

The markdown for nested ordered lists is:

~~~markdown
1. Prepare the environment
   1. Install dependencies
   2. Configure settings
2. Run the application
   1. Start the server
   2. Verify the output
~~~

1. Prepare the environment
   1. Install dependencies
   2. Configure settings
2. Run the application
   1. Start the server
   2. Verify the output

## Mixed Nested Lists

The markdown for mixed nested lists is:

~~~markdown
1. First ordered item
   - Unordered sub-item
   - Another sub-item
2. Second ordered item
   - Sub-item here
~~~

1. First ordered item
   - Unordered sub-item
   - Another sub-item
2. Second ordered item
   - Sub-item here

## Task Lists

The markdown for task lists is:

~~~markdown
- [x] Completed task
- [x] Another completed task
- [ ] Incomplete task
- [ ] Another incomplete task
~~~

- [x] Completed task
- [x] Another completed task
- [ ] Incomplete task
- [ ] Another incomplete task
