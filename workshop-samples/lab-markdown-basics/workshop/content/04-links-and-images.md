---
title: Links and Images
---

Markdown supports inline links, reference-style links, and images.

## Inline Links

The markdown for an inline link is:

~~~markdown
Visit [Example Website](https://www.example.com) for more information.
~~~

Visit [Example Website](https://www.example.com) for more information.

## Links with Titles

The markdown for a link with a hover title is:

~~~markdown
Visit [Example Website](https://www.example.com "Example Site") to see the title on hover.
~~~

Visit [Example Website](https://www.example.com "Example Site") to see the title on hover.

## Reference-Style Links

The markdown for reference-style links is:

~~~markdown
See the [Educates documentation][educates-docs] for details.

[educates-docs]: https://docs.educates.dev "Educates Documentation"
~~~

See the [Educates documentation][educates-docs] for details.

[educates-docs]: https://docs.educates.dev "Educates Documentation"

## Autolinked URLs

The markdown for an autolinked URL is:

~~~markdown
<https://www.example.com>
~~~

<https://www.example.com>

## Images

The markdown for an image is:

~~~markdown
![Placeholder Image](https://avatars.githubusercontent.com/u/72636080?s=128&v=4)
~~~

![Placeholder Image](https://avatars.githubusercontent.com/u/72636080?s=128&v=4)

## Images with Links

The markdown for a clickable image is:

~~~markdown
[![Linked Image](https://avatars.githubusercontent.com/u/72636080?s=128&v=4)](https://www.example.com)
~~~

[![Linked Image](https://avatars.githubusercontent.com/u/72636080?s=128&v=4)](https://www.example.com)
