---
title: Blockquotes and Horizontal Rules
---

Blockquotes are used to highlight quoted text or call attention to a passage.
Horizontal rules create visual dividers between sections.

## Blockquotes

The markdown for a blockquote is:

~~~markdown
> This is a blockquote. It can span multiple lines and is typically used to
> highlight important information or quote someone.
~~~

> This is a blockquote. It can span multiple lines and is typically used to
> highlight important information or quote someone.

## Nested Blockquotes

The markdown for nested blockquotes is:

~~~markdown
> This is an outer blockquote.
>
> > This is a nested blockquote inside the outer one.
>
> Back to the outer blockquote.
~~~

> This is an outer blockquote.
>
> > This is a nested blockquote inside the outer one.
>
> Back to the outer blockquote.

## Blockquotes with Other Elements

The markdown for a blockquote containing other markdown elements is:

~~~markdown
> **Note:** You can use other markdown inside blockquotes.
>
> - List item one
> - List item two
>
> ```
> Even code blocks work inside blockquotes.
> ```
~~~

>

>
> ```

> ```

## Horizontal Rules

The markdown for horizontal rules is:

~~~markdown
Content above the rule.

---

Content between rules.

***

Content below the rules.
~~~

Content above the rule.

---

Content between rules.

***

Content below the rules.
