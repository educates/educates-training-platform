title: Workshop Summary
---

This workshop demonstrated common markdown formatting as rendered by Educates.

**Markdown elements covered:**

- Headings — Levels `h1` through `h6` using `#` syntax
- Text formatting — Bold, italic, bold+italic, strikethrough, and inline code
- Lists — Unordered, ordered, nested, mixed, and task lists
- Links — Inline links, links with titles, reference-style links, and autolinks
- Images — Basic images and clickable images with links
- Code blocks — Fenced code blocks with syntax highlighting for Python, YAML, Bash, JSON, and plain text
- Blockquotes — Single and nested blockquotes with mixed content
- Horizontal rules — Visual dividers using `---` and `***`
- Tables — Basic tables, column alignment, and tables with inline formatting

**Admonition shortcodes covered:**

- `{{</* note */>}}` — Blue informational callout with optional custom title
- `{{</* warning */>}}` — Yellow caution callout with optional custom title
- `{{</* danger */>}}` — Red critical warning callout with optional custom title
